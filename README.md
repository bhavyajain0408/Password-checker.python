# Password Strength Checker 🔐

A simple Python-based Password Strength Checker that analyzes password security using multiple validation rules such as length, uppercase letters, lowercase letters, numbers, and special characters.

## Features

* Checks minimum password length
* Detects uppercase and lowercase letters
* Validates numeric digits
* Detects special characters
* Provides strength feedback (Weak, Medium, Strong)
* User-friendly command-line interface

## Technologies Used

* Python
* Regular Expressions (`re` module)

## How It Works

The program continuously asks users to enter a password and evaluates its strength based on predefined security conditions. Users can type `exit` anytime to quit the application.

## Example

Enter your password: Hello123
Output: Medium: Add special characters to make your password stronger.

Enter your password: Hello@123
Output: Strong: Your password is secure!
